How to Use:

  1. Important: Must use local server (not file://)
    Run python -m http.server 8080 then open http://localhost:8080
  2. Wait for both models to load (Whisper + LLM dots turn green)
  3. Click "Start Recording"
  4. Select a browser tab with Teams/Zoom and enable "Share tab audio"
  5. Allow microphone access when prompted
  6. Both meeting audio + microphone will be captured
  7. Click "Export Markdown" to download meeting notes
